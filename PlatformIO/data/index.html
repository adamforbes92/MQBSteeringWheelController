Live bus status. CAN and LIN read Healthy while valid frames are being received. LIN 2 shows Idle when the chassis bus is not being polled. Resistive shows the current wiper resistance.

PNP Output (Pin 21)

Drives GPIO 21 HIGH to test the PNP transistor output circuit.

Resistive Output (Aux)

Holds the X9C10X digital potentiometer at the selected resistance for bench testing. Overrides button-triggered momentary outputs.

Test Resistance

Test value: 0 Ω
Each press steps the test value by 200 Ω, pulses the output DOWN to that value for 0.5 s, then returns it to idle HIGH. Use this to probe which resistances the stereo responds to.
